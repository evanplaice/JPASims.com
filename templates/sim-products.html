TCAS (Traffic Collision Avoidance System)
The Simulated TCAS system is designed as a "Bolt On" system with all hardware components contained in the Industrial PC and Special TCAS Interface Unit.
This arrangment:
  • Allows easy installation and integration into the Simulator with no Training downtime
  • Provides a complete COTS (Commercial Off The Shelf) hardware package
  • Provides an Industrial PC platform which can be utilized to host other simulator systems (EGPWS, GPS, Radio Aids Compiler, QTG System, etc.)
This system is currently operating on 25 simulators worldwide. The Latest TCAS II Version 7 is now available. The high time system has operated for over 24,000 hours without a single failure.

RIOS (Replacement Instructor Operating System)
A new JPA product, the Replacement Instructor Operator System (RIOS) was launched at the recent FSEMC Conference in Montreal. A dual touchscreen demonstration system was on display interacting with a "quasi-simulation" program running to demonstration integration to a host computer. The RIOS is a low cost COTS (Commercial Off-The-Shelf) solution which is capable of replacing any IOS system operating today.
